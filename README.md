# Математически речник 5.–7. клас

Интерактивен HTML речник с основни термини и кратки дефиниции по математика за прогимназиален етап.

## Какво има вътре

- търсене по термин и дефиниция
- филтър по клас: 5., 6. и 7. клас
- филтър по тема
- азбучен филтър
- разгъваеми карти с кратки дефиниции
- мобилен изглед

Сайтът е в `index.html` и е подходящ за GitHub Pages.
